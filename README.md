# Customer Churn Analysis

## 📊 Project Overview
This project analyzes customer churn to uncover the factors that influence whether customers stay or leave. Using Excel as the primary analysis tool, the project includes data cleaning, exploratory data analysis (EDA), feature exploration, and visual storytelling to highlight the most important churn drivers.

The goal is to help organizations better understand customer behavior and develop targeted retention strategies.

---

## 🧰 Tools & Technologies
- **Microsoft Excel**
  - Data cleaning and preprocessing
  - Pivot tables and aggregations
  - Exploratory data analysis
  - Charts and dashboards

---

## 🔍 Key Objectives
- Identify patterns and trends associated with customer churn  
- Explore demographic, behavioral, and service‑related variables  
- Visualize churn distribution and high‑risk customer segments  
- Provide actionable insights to support retention efforts  

---

## 📈 Analysis Highlights
Typical insights included in this project:
- Churn rate calculation and distribution  
- Comparison of churn vs. non‑churn customers  
- Impact of contract type, tenure, monthly charges, and service usage  
- Identification of high‑risk customer groups  
- Dashboard summarizing key metrics and visualizations
